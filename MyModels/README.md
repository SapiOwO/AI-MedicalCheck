# Medical Check - Emotion Detection Service

Tempatkan file model PyTorch (.pth) di folder ini.

## File yang dibutuhkan:
- `emotion_best.pth` - Model untuk emotion detection

## Cara menggunakan:
1. Copy file .pth model kamu ke folder ini
2. Python service akan otomatis load model saat startup
3. Tidak perlu restart Laravel

## Note:
Jika kamu punya model lain (fatigue_best.pth, pain_best.pth), bisa ditambahkan nanti.
